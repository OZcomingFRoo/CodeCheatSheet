Default Page (Home)

This project contains the following technologies/libraries

Bootstrap 4

One of the most common CSS class libraries to use for styling the page without using inline CSS and creating on the fly CSS classes for common styling

Angular (11)

The main SPA framework used to creat big scale web application with many tools out of the box such as:

  • Routing
  • Http Calls
  • A component-based framework

Jasmine JS

This comes in addition with Angular used for Unit/integration testing.

Note! There's also an option for E2E testing.
